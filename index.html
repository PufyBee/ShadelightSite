Get Shadelight

Install locally and scan your network or files in seconds.

git clone https://github.com/PuffyBee/Shadelight.git
cd shadelight
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate
pip install -r requirements.txt
python -m shadelight 192.168.1.0/24 --ports 22,80,443
